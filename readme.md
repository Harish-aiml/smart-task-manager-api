# Smart Task Manager API

A backend task management system built using FastAPI, SQLite, and SQLAlchemy.

## Features

- Create tasks
- View all tasks
- Update tasks
- Delete tasks
- Search tasks by title
- Filter completed tasks
- Filter pending tasks
- Priority levels
- Due dates

## Tech Stack

- FastAPI
- Python
- SQLite
- SQLAlchemy

## Run Project

```bash
uvicorn main:app --reload
